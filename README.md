# Customer Support Ticket Analysis

This capstone case study analyzes a Kaggle customer support ticket dataset to better understand ticket volume, ticket priority, support channels, and customer satisfaction.

## Business Question
How do ticket type, ticket priority, and support channel relate to customer satisfaction and ticket volume?

## Tools Used
- Python/Pandas
- SQL
- Data visualization
- Dashboard charts

## Key Findings
- The dataset included 8,469 rows and 17 columns.
- Satisfaction analysis focused on closed tickets because those were the records with satisfaction ratings.
- Chat had the highest average satisfaction rating.
- Low priority tickets had the highest average satisfaction rating.
- Missing satisfaction ratings and resolution-time issues were important data quality concerns.

## Recommendations
- Track satisfaction ratings by ticket type.
- Review high-priority and critical tickets more closely.
- Compare satisfaction across email, chat, phone, and social media support.
- Improve data collection for resolution time and satisfaction ratings.
